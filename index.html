Booklend

  • GitHub Repository

As a side project, I am building a book trading application using React, Node, Typscript and GraphQL. The app uses Google authentication for user signup and login, Google's Geo-location API to allow searching for books, and Cloudinary API to store images. GraphQL requests are handled by React Apollo and the Apollo CLI. Finally we use, MongoDB to persist data. It is still a work-in-progress and, using the help of NewLine's Fullstack React Masterclass, I will continue to build and improve the app.

One Time Pads

  • GitHub Repository

Five small C programs that encrypt and decrypt information using a one-time pad-like system. These programs combine multi-processing code with socket-based inter-process communication. Two of these will programs function as servers, and will be accessed using network sockets. Two will be clients and each one will use one of the servers to perform cipher and decipher tasks. The keygen will generate a key of specificed length to use for the encryptiong and decryption process.

Soapbox

  • GitHub Repository

As part of our Capstone course project, I am currently working with a partner on a mobile based, geo-fenced social media platform. The app, Soapbox, will allow users to post a message, which will be known as a "shout." The shout will be geo-fenced within a certain radius. The shout can be viewed by any other user within the radius but as soon as the user leaves the radius, the shout will no longer be viewable to them. The shouts will also expire within a certain time limit. This project, still a work-in-progress, will allow us to advance our skills in iOS development and programming in Swift.
